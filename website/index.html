The Web4 Browser

An open-source, AI-native browser where intelligent agents are first-class citizens.

What is Web4?

Agentic

AI as First-Class Citizen

AI agents aren't add-ons — they're core to the browsing experience. The browser is built from the ground up for autonomous, intelligent operation.

Decentralized

Compute Equity & Sovereignty

Your data stays yours. Decentralized architecture ensures compute equity, privacy by default, and sovereignty over your digital footprint.

Crypto-Native

Settlement Protocol for Machines

Built-in wallet and crypto settlement layer enables machine-to-machine transactions, powering an autonomous agent economy.

Key Features

Agentic Browser

Intelligence Built In

  • AI-Native Kernel — reasoning engine at the core, not bolted on
  • Autonomous Task Execution — agents browse, fill, click, extract on your behalf
  • Self-Healing Workflows — agents adapt when pages change or errors occur
Decentralized & Private

Your Data, Your Rules

  • Freedom of Choice — pick your AI provider, model, and data policy
  • Always-On — local-first architecture works without cloud dependency
  • Chrome Lossless — full compatibility with existing extensions & sites
Crypto-Ready

The Agent Economy

  • Agent Economy — agents can pay, earn, and settle autonomously
  • Wallet-Native — built-in wallet for seamless crypto interactions
  • Open & Extensible — plugin system for community-built capabilities

Get Started

terminal
# Clone the repository
$ git clone https://github.com/instry/ocbot.git
$ cd ocbot
 
# Install and run
$ npm install
$ npm start
