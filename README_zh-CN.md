# ocbot - AI Browser & Assist

## **[https://oc.bot](https://oc.bot)**

[English](./README.md) | **中文**

---

**ocbot** 是一个 AI 原生浏览器，也是你的私人 AI 助理。你可以理解为它是 **Chrome + Gemini** 的开源版，也是 **OpenClaw** 的集成版。

## ✨ 核心特性 (Key Features)

### 1. AI 浏览器 (AI Browser)
*   **Chrome + Gemini 开源实现**：AI 能力深度植入浏览器内核，而非插件。
*   **模型自由**：你可以自由切换大语言模型。支持云端主流模型，也支持完全**本地运行的 LLM**，确保数据隐私与控制权。
*   **Chrome无损体验**：完整保留 Chrome 使用体验。支持一键导入 Chrome 数据（书签、历史记录、密码），让切换变得毫无门槛。

### 2. AI 助理 (AI Assistant)
*   **类似 OpenClaw 的体验**：ocbot 不仅运行在桌面，还能通过手机浏览器或第三方聊天软件（IM）远程交互。
*   **随时待命**：无论身在何处，都可以远程指挥家中的浏览器处理复杂任务。
*   **移动端**：提供原生的 iOS/Android 浏览体验，与桌面端无缝同步标签页和历史记录，没有任何割裂感。

### 3. 取代 RPA (RPA Killer)
无需编码，通过自然语言即可实现网页自动化：
*   **自然语言控制**：直接告诉浏览器 "帮我下载所有发票" 或 "监控机票价格"。
*   **一次学习，多次复用**：AI 学习一次操作流程后可转化为高效执行路径，后续运行**大幅节省 Token 成本**。
*   **自愈能力 (Self-Healing)**：网页 UI 变了？AI 能通过视觉理解自动修复操作路径，确保持续稳定运行。
