# Extension UI Internationalization (i18n) Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add multi-language support (en + zh-CN) to the Ocbot extension UI, defaulting to system language with runtime manual switching.

**Architecture:** Lightweight custom i18n — locale JSON files loaded at runtime via React Context + `useI18n()` hook. The existing `useSettings` hook stores language preference. System language detected via `chrome.i18n.getUILanguage()` on first launch.

**Tech Stack:** React Context, chrome.i18n.getUILanguage(), chrome.storage.local

### Task 2: Create i18n module (context + hook)

**Files:**
- Create: `web/lib/i18n/index.ts`
- Create: `web/lib/i18n/context.tsx`

**Step 1: Create the core i18n module**

`web/lib/i18n/index.ts` — Loads locale data and provides a `t()` function:

```typescript
import en from './locales/en.json'
import zh from './locales/zh.json'

export type Locale = 'en' | 'zh'

const messages: Record<Locale, Record<string, string>> = { en, zh }

/**
 * Detect system language, mapped to supported locales.
 * Falls back to 'en'.
 */
export function detectSystemLocale(): Locale {
  try {
    const uiLang = chrome.i18n?.getUILanguage?.() ?? navigator.language
    if (uiLang.startsWith('zh')) return 'zh'
    return 'en'
  } catch {
    return 'en'
  }
}

/**
 * Create a translation function for a given locale.
 * Supports {key} interpolation: t('chat.actionsCompleted', { count: 3 })
 */
export function createT(locale: Locale) {
  const dict = messages[locale] ?? messages.en
  const fallback = messages.en

  return function t(key: string, params?: Record<string, string | number>): string {
    let text = dict[key] ?? fallback[key] ?? key
    if (params) {
      for (const [k, v] of Object.entries(params)) {
        text = text.replace(new RegExp(`\\{${k}\\}`, 'g'), String(v))
      }
    }
    return text
  }
}
```

**Step 2: Create React context + hook**

`web/lib/i18n/context.tsx`:

```tsx
import { createContext, useContext, useMemo, type ReactNode } from 'react'
import { createT, type Locale } from './index'

interface I18nContextValue {
  locale: Locale
  t: (key: string, params?: Record<string, string | number>) => string
}

const I18nContext = createContext<I18nContextValue>({
  locale: 'en',
  t: (key) => key,
})

export function I18nProvider({ locale, children }: { locale: Locale; children: ReactNode }) {
  const value = useMemo(() => ({
    locale,
    t: createT(locale),
  }), [locale])

  return <I18nContext.Provider value={value}>{children}</I18nContext.Provider>
}

export function useI18n() {
  return useContext(I18nContext)
}
```
